# v0.1.0 Release Notes

`v0.1.0` is the first public-preview release of DCF Dashboard.

## Highlights

- Working browser-based DCF demo with a mock-backed UI path
- Direct FastAPI compute flow with a reproducible sample payload
- Optional Monte Carlo output for percentile-based valuation ranges
- Optional Convex persistence for saved runs and replay history
- Public-facing governance, roadmap, contribution, and security docs

## Setup Notes

- Use Node `22.x`, npm `11.x`, and Python `3.12+`
- Run `npm ci` for JavaScript dependencies
- Run `python -m pip install -r python/requirements-dev.txt -c python/constraints.txt` inside a virtualenv for Python dependencies
- `npm` is canonical; Bun is used only through the npm test scripts

## Breaking Changes

None in this release.

## Known Limitations

- The UI demo is mock-backed by default and is designed to prove the interaction model quickly
- Convex-backed persistence is optional and still best suited for maintainers or advanced contributors
- Deployment guidance exists, but hosted production operation is not the primary focus of `v0.1.0`
