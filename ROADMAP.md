# Roadmap

This roadmap is intentionally short and public-facing. It focuses on the next meaningful improvements for contributors and evaluators.

## Current Milestone: Public Preview Follow-Up

- Tighten onboarding for first-time contributors
- Expand sample valuation cases beyond the default demo payload
- Document the Convex persistence flow end to end
- Improve screenshots, demos, and product proof in the repo
- Keep security and dependency audits current as the public surface grows

## Next Up

- Add more explainers around model assumptions and DCF caveats
- Improve replay/history UX for saved runs
- Expand test coverage around Next.js to FastAPI integration paths
- Publish clearer deployment guidance for optional Convex-backed setups

## Not In Scope Right Now

- Multi-tenant SaaS features
- Real-time market data integrations as a default requirement
- Packaging the project as a hosted service before the local and contributor story is stronger
