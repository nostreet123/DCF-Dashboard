# Code Of Conduct

## Our Standard

This project aims to be welcoming, respectful, and technically rigorous. Contributors are expected to:

- be constructive and kind
- assume good intent while discussing tradeoffs
- give actionable technical feedback
- avoid harassment, discrimination, and personal attacks

## Unacceptable Behavior

- abusive or insulting language
- harassment or intimidation
- publishing private information without permission
- repeated bad-faith disruption of discussions or review

## Enforcement

Project maintainers may edit, remove, or reject comments, issues, pull requests, and other contributions that do not meet this standard.

## Reporting

If you experience or observe unacceptable behavior, contact the maintainer privately through GitHub before escalating publicly.
